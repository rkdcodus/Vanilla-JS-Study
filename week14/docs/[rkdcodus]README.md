# Grocery Bud

- CRUD 기능이 있는 장바구니 리스트이다.
- 장바구니 리스트에 아이템을 추가하고 삭제할 수 있다.
- 아이템명을 수정할 수 있다.
- 리스트에 담긴 아이템들을 한번에 삭제할 수 있다.
